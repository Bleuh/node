### Comment démarrer le projet
* **Cloner le projet**
`git clone https://github.com/Bleuh/node`
* **Demarer nodejs**
Allez dans le dossier */node* puis lancer la commande suivante :
 `npm start`
* **Le serveur est sur le port 8080**
`localhost:8080`
